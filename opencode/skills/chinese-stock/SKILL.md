# 中国A股信息查询

## 触发条件
用户问题涉及以下场景时激活本 Skill：
- A股股票行情查询
- 股票代码、股价、涨跌幅
- 成交量、成交额、市值
- 基本面数据（市盈率、市净率、ROE等）
- 技术指标（MACD、KDJ、布林带等）
- 龙虎榜、机构持仓
- 新股申购、日历
- 板块行情、行业走势
- 大盘指数（上证指数、深证成指、创业板指等）

---

## 工作流程 （严格按顺序执行，不得跳过）

### 第一步：信息收集

检查用户是否提供了以下信息：

**必问项（缺失时必须追问）：**
1. 📌 股票代码/名称（如 600519、茅台）
2. 📋 查询类型（实时行情/基本面/技术指标/龙虎榜/新股/板块）

**选问项：**
3. 📅 时间范围（当日/近一周/近一月/近一年）
4. 📊 数据维度（仅价格/完整行情/财务数据/技术面）

追问格式：
> 请告诉我股票代码或名称，以及想查询什么类型的数据？（行情/基本面/技术面/龙虎榜/新股）

---

### 第二步：获取数据

使用以下数据源获取信息：

| 数据类型 | 数据源 | 说明 |
|---------|--------|------|
| 实时行情 | 新浪财经/东方财富API | 股价、涨跌幅、成交量 |
| 基本面 | 新浪财经/A班港 | 市盈率、市净率、ROE |
| 技术指标 | 东方财富/同花顺 | MACD、KDJ、布林带、均线 |
| 龙虎榜 | 东方财富 | 游资/机构买卖、营业部 |
| 资金流向 | 东方财富 | 主力资金流入流出 |
| 盘口数据 | 东方财富 | 买卖五档、逐笔成交 |
| 新股日历 | 东方财富 | 申购日期、上市日期、中签率 |
| 板块行情 | 东方财富 | 行业板块、概念板块 |

**API接口示例（免费）：**
```
# 新浪股票接口
https://hq.sinajs.cn/list=sh600519

# 东方财富行情
http://push2.eastmoney.com/api/qt/ulist.np/get

# 龙虎榜数据
http://push2ex.eastmoney.com/getTopicZDFenBu
```
