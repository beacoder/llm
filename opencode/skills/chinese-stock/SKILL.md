# 中国A股信息查询

---

## 工作流程（严格按顺序执行，不得跳过）

### 第一步：信息收集

开始查询前，检查用户是否提供了以下信息。**缺失关键信息时必须追问，不得自行假设。**

**必问项：**
1. 📌 股票代码/名称（如 600519、茅台）
2. 📋 查询类型（实时行情 / 基本面 / 技术指标 / 龙虎榜 / 新股 / 板块 / 资金流向）

**选问项：**
3. 📅 时间范围（当日 / 近一周 / 近一月 / 近一年）
4. 📊 数据维度（仅价格 / 完整行情 / 财务数据 / 技术面）

追问格式（简洁）：

> 请告诉我股票代码或名称，以及想查询什么类型的数据？（行情 / 基本面 / 技术面 / 龙虎榜 / 新股）

---

### 第二步：多源数据获取

按以下数据源依次获取，每个数据源标注来源：

| 数据类型 | 数据源 | 说明 |
|---------|--------|------|
| 实时行情 | 新浪财经 / 东方财富 API | 股价、涨跌幅、成交量 |
| 基本面 | 新浪财经 / A 股班 | 市盈率、市净率、ROE |
| 技术指标 | 东方财富 / 同花顺 | MACD、KDJ、布林带、均线 |
| 龙虎榜 | 东方财富 | 游资/机构买卖、营业部 |
| 资金流向 | 东方财富 | 主力资金流入流出 |
| 盘口数据 | 东方财富 | 买卖五档、逐笔成交 |
| 新股日历 | 东方财富 | 申购日期、上市日期、中签率 |
| 板块行情 | 东方财富 | 行业板块、概念板块 |

**API 接口示例（免费）：**
```
# 新浪股票接口
https://hq.sinajs.cn/list=sh600519

# 东方财富行情
http://push2.eastmoney.com/api/qt/ulist.np/get

# 龙虎榜数据
http://push2ex.eastmoney.com/getTopicZDFenBu
```

**获取规则：**
- 每个数据源标注来源与更新时间
- 不可访问则标记「无法获取」后继续下一个
- 优先使用实时数据，缓存数据需标注延迟

---

### 第三步：整理与校验

- 合并多源数据，去除冲突条目
- **交叉验证：** 多源一致的数据标记为「✅ 高可信」，单源数据标记为「💬 仅供参考」
- 科创板 / 创业板股票单独标注板块
- ST 股票需醒目提示风险
- 停牌 / 退市股票需标注状态

---
